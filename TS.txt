Реализовать калькулятор для поточного выполнения арифметических операций.

История 1: как пользователь, я хочу произвести вычисления так,
             что бы получить их результаты с указанием типа операции,
             которая была произведена в каждом вычислении, независимо от количества вычислений;

Техническое задание.

Реализовать абстрактный механизм выполнения арифм. операции над 2-мя числами;
Реализовать механизм выбора типа операции, исходя из значения технической строки;

Все вычисления должны выполняться в потоке (выполнить все операции в рамках одного цикла);
Предусмотреть возможности сортировки по типу операций и по их результатам;
Число операций должно быть не менее 10;

Результаты вычислений вывести на консоль в формате: [ тип операции : значение ]

Технические ограничения:
* выполять вычисления над любыми наследниками класса Number;
* принимаемые данные находятся в Map<String, Number>;
* возвращаемые данные находятся в LinkedHashMap<String, Number>.

Технологии:

Использовать возможности java 8+;
Использовать функционал классов: Enum, Function (собственная релизация), Stream (в рамках цикла), Collectors;
